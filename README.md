# Mini-stry (Policy Runtime Platform)

Mini-stry is a domain-neutral Policy Runtime Platform.

## Tech Stack

- Convex 1.39.1
- TypeScript 5.9.3
- Vitest 4.1.7
- ESLint 10.4.1 (flat config)

## Quick Start

```bash
npm install
npm test
npm run lint
npm run typecheck
```

See [ARCHITECTURE.md](ARCHITECTURE.md) for the Module Boundary Rule.
