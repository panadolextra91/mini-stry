# Approval Module (Reference Decision Consumer)

This module implements the Approval Routing reference decision consumer (DEC-03). It is responsible for translating `request-approval` Decisions generated by the Policy Runtime into sequential, multi-stage approval chains.

## Purity Rules & Architecture
1. **No Downstream Dependency (SC#1):** The core Policy Runtime (`modules/request/`, `modules/runtime/`) has ZERO compile-time dependency on this module. It operates purely as a subscriber to decisions.
2. **Barrel-Only Imports (D-08):** Cross-module imports must only target this module's public barrel (`index.ts`). Deep imports into internal files are blocked by ESLint.

## Directory Layout
* `domain/`: Pure domain types, entities (ApprovalChain, ApprovalTask), and the pure task state machine.
* `application/`: Application-layer error classes.
* `ports/`: Outgoing repository ports (ApprovalChainRepositoryPort, ApprovalTaskRepositoryPort).
* `adapters/`: Outbound adapter implementations (Memory adapters).
