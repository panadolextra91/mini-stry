    Create `src/modules/policy/application/policy-service.ts` with `PolicyService` class. Constructor takes: `policyRepo: PolicyRepositoryPort`, `versionRepo: PolicyVersionRepositoryPort`, `validator: SchemaValidatorPort`, `dispatcher: EventDispatcher<PolicyEventMap>`. Every method takes `ctx: TenantContext` as first parameter (TenantContext constraint).

    Methods:
    - `createPolicy(ctx, input: { name: string }): Promise<Policy>` — delegates to `policyRepo.create()`.
    - `createDraft(ctx, policyId: PolicyId, content: unknown, createdBy: UserId): Promise<PolicyVersion>` — checks no existing draft via `versionRepo.findDraftByPolicy()`; throws `DraftAlreadyExistsError` if one exists (D-32 one-draft rule). Gets next version number. Runs `validator.validate(content)` to set initial `validationStatus`/`validationErrors` (D-34). Creates version with `status='draft'`, `revision=0`. Emits `DraftCreated` event. Returns the new draft.
    - `saveDraft(ctx, versionId: PolicyVersionId, content: unknown, expectedRevision: number): Promise<PolicyVersion>` — finds version, throws `DraftNotFoundError` if missing, `ImmutableVersionError` if `status='published'` (POL-03). Checks `version.revision !== expectedRevision` → throws `ConflictError` (D-36). Runs `validator.validate(content)` to update `validationStatus`/`validationErrors`. Updates via `versionRepo.update()` with `revision + 1`. Emits `DraftUpdated`. Returns updated version.
    - `publishDraft(ctx, versionId: PolicyVersionId): Promise<PolicyVersion>` — finds version, throws `DraftNotFoundError` if missing, `ImmutableVersionError` if already published. Checks `validationStatus !== 'valid'` → throws `InvalidPublishError` with stored `validationErrors` (D-34). Updates: `status='published'`, `publishedAt=Date.now()`. Updates `Policy.activeVersionId` via `policyRepo.updateActiveVersion()`. Emits `PolicyPublished`. Returns published version.

    Create in-memory fakes following `InMemoryRoleRepository` pattern:
    - `src/modules/policy/adapters/memory/in-memory-policy-repository.ts` — `Map<string, Policy>` keyed by id, filtered by `ctx.tenantId`.
    - `src/modules/policy/adapters/memory/in-memory-policy-version-repository.ts` — `Map<string, PolicyVersion>` keyed by id, `findDraftByPolicy` filters by `policyId + status='draft'`, `getNextVersionNumber` returns max+1.

    - **publishDraft**: transitions valid draft to published with publishedAt set; updates Policy.activeVersionId; throws InvalidPublishError for invalid draft (D-34); throws ImmutableVersionError for already-published version; emits PolicyPublished event.
    - **immutability (POL-03)**: saveDraft on published version throws ImmutableVersionError; publishDraft on published version throws ImmutableVersionError.
