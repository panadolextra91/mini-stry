## Objective

Set up the modular modular-monolith Hexagonal folder structure, configure the Vitest environment for unit testing, and implement the core tenant and user domain entities and role systems in Pure TypeScript.

1. Modular directories under `src/modules/tenant/` exist and match the Hexagonal Architecture guidelines.
