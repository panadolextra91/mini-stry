# Plan 01-03: Implement AST Interpreter, Runtime Services, and Vitest Suites

## Objective

Develop the secure AST Interpreter (evaluator) in pure TS, write the PolicyRuntimeService orchestrating compilers, and build the Vitest unit testing suite achieving 100% logic test coverage.

## Verification Criteria

1. `ASTInterpreter` evaluates AST structures against dynamic payload variables correctly.
2. `PolicyRuntimeService` parses DSL inputs and returns correct decisions.
3. Unit tests achieve 100% code coverage.
4. `npx vitest run policy-runtime.service.spec.ts` executes successfully with 0 errors.

## must_haves

1. DSL interpreter must evaluate conditional parameters securely without any dynamic code compilation (`eval()` ban).
