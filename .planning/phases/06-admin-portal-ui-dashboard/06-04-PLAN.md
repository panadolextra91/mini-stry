<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| editor (advisory) → server validator (authoritative) | Monaco markers are advisory; PolicyService Ajv is the source of truth on publish (D-58) |
| client policy JSON → backend | Untrusted policy content; validated server-side, never executed |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-10 | Tampering | publish path | mitigate | Server Ajv (PolicyService.publishDraft) is authoritative; Monaco is advisory only; invalid drafts cannot be published (D-58) |
| T-06-11 | Tampering / Elevation | policy JSON content | mitigate | Policy text is JSON.parse'd and structurally evaluated by the runtime; never eval'd (RUN-02). Rendered as editor/text, never dangerouslySetInnerHTML |
| T-06-12 | Tampering | Monaco schema source | mitigate | Single canonical schema (D-59) fed to Monaco; no editor-specific schema defined; parity test (06-01) guards divergence |
</threat_model>
